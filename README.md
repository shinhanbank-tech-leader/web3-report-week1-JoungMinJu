# 제목: 블록체인의 신뢰 형성 구조와 기존 금융 시스템의 신뢰 구조 비교

## 부서명: Data플랫폼Unit
## 이름: 정민주

-------------------

## 레포트 내용
전통적인 금융은 사람, 기관이라는 **"신뢰할 수 있는 제 3자"**에 의하여 신뢰구조가 형성됨.
하지만 블록체인의 경우 **분산합의**, 그리고 그것이 구현된 **알고리즘**에 대한 신뢰를 기반으로 신뢰구조가 형성된다.
수천 개의 노드가 동일한 원장을 분산 보유하고, 새로운 거래는 네트워크 전체의 합의를 거쳐야만 기록되는 구조로 수학과 코드에 신뢰를 내재화한 것이다.

따라서 전통 금융은 중앙집중형의 신뢰구조인 반면, 블록체인은 수평적인 구조의 분산형 신뢰구조라 말할 수 있다.
